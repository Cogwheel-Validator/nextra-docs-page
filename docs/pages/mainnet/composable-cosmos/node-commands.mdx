## üßô Validator management



<Callout type="info" emoji="üôã‚Äç‚ôÄÔ∏è">
  Make sure to change values for moniker, identity, details website to your own values.
</Callout>

#### Edit existing validator

```bash
{$BINARY} tx staking edit-validator \
--new-moniker "YOUR_MONIKER_NAME" \
--identity "YOUR_KEYBASE_ID" \
--details "YOUR_DETAILS" \
--website "YOUR_WEBSITE_URL" \
--chain-id {$CHAIN_ID} \
--commission-rate 0.05 \
--from wallet \
--gas-adjustment 1.5 \
--gas auto \
--gas-prices 0{$DENOM} \
-y
```

#### Unjail validator

```bash
{$BINARY} tx slashing unjail --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto
 --gas-prices 0{$DENOM} -y
```

#### Jail reason

```bash
{$BINARY} query slashing signing-info $( {$BINARY} tendermint show-validator)
```

#### List all active validators

```bash
{$BINARY} q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_BONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### List all inactive validators

```bash
{$BINARY} q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_UNBONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### View validator details

```bash
{$BINARY} q staking validator $( {$BINARY} keys show wallet --bech val -a)
```

## ü™ô Token management

#### Withdraw rewards from all validators

```bash
{$BINARY} tx distribution withdraw-all-rewards --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto --gas-prices 0{$DENOM} -y
```

#### Withdraw commission and rewards from your validator

```bash
{$BINARY} tx distribution withdraw-rewards $( {$BINARY} keys show wallet --bech val -a) --commission --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto --gas-prices 0{$DENOM} -y
```

#### Delegate tokens to yourself

```bash
{$BINARY} tx staking delegate $( {$BINARY} keys show wallet --bech val -a) 1000000000000{$DENOM} --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto --gas-prices 0{$DENOM} -y
```

#### Delegate tokens to validator

```bash
{$BINARY} tx staking delegate <TO_VALOPER_ADDRESS> 1000000000000{$DENOM} --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto --gas-prices 0{$DENOM}  -y
```

#### Redelegate tokens to another validator

```bash
{$BINARY} tx staking redelegate $( {$BINARY} keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000${CHAIN_DENOM} --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto --gas-prices 0{$DENOM} -y
```

#### Unbond tokens from your validator

```bash
{$BINARY} tx staking unbond $( {$BINARY} keys show wallet --bech val -a) 1000000${CHAIN_DENOM} --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto --gas-prices 0{$DENOM} -y
```

#### Send tokens to the wallet

```bash
{$BINARY} tx bank send wallet <TO_WALLET_ADDRESS> 1000000${CHAIN_DENOM} --from wallet --chain-id {$CHAIN_ID} --gas-adjustment 1.5 --gas auto --gas-prices 0{$DENOM} -y
```

## ‚ö°Ô∏è Utility

#### Update Indexer

**Disable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "null"|' {$CHAIN_HOME}/config/config.toml
```

**Enable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "kv"|' {$CHAIN_HOME}/config/config.toml
```

#### Update pruning

```bash
sed -i \
  -e 's|^pruning *=.*|pruning = "custom"|' \
  -e 's|^pruning-keep-recent *=.*|pruning-keep-recent = "2000"|' \
  -e 's|^pruning-keep-every *=.*|pruning-keep-every = "0"|' \
  -e 's|^pruning-interval *=.*|pruning-interval = "100"|' \
  {$CHAIN_HOME}/config/app.toml
```
